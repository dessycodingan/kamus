Application Programming Interface (API)

Artian dari perolehan akses suatu aplikasi ke resource sistem, yang biasanya digunakan untuk tujuan komunikasi (pengiriman dan penerimaan data), perolehan data kembali atau layanan sistem lainnya. Dalam area emulasi terminal yang lebih khusus, API menyediakan simulasi penekanan kunci dan menulis di / membaca dari ruang presentasi (device buffer). Ia dapat juga menyediakan pengiriman dan penerimaan field terstruktur.
