Buffer

Area memori atau register elektronis dimana data disimpan sementara sembari menunggu letak penempatannya. Buffer mengimbangi perbedaan dalam data-flow rate (misalnya antara suatu terminal dan line transmisinya). Buffer juga digunakan sebagai mekanisme backup data untuk “memegang” data yang mungkin akan ditransfer kembali jika terjadi kesalahan selama proses transmisi.
