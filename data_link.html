Data Link Layer

Layer kedua dari model OSI. Data Link Layer bertanggung jawab untuk penyediaan transmisi data melalui link tunggal dari satu sistem ke sistem lain. Mekanisme kontrol dalam Data Link Layer menangani transmisi data unit, yang sering disebut frame, melalui suatu circuit fisik. Fungsi-fungsi yang beroperasi pada Data Link Layer memungkinkan data ditransmisikan, dalam mode yang relatif bebas dari kesalahan, melalui suatu circuit fisik yang kadang kala cenderung memiliki kesalahan. Layer ini mengurus bagaimana bit-bit dikumpulkan kedalam frame-frame dan mengimplementasikan mekanisme pendektesian error yang mengidentifikasikan transmission errors. Pada beberapa tipe data link. Data Link layer juga dapat membentuk prosedur-prosedur untuk flow-control, frame sequencing dan pemulihan kesalahan transmisi.
