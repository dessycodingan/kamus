Application Interface

Himpunan software routine dan perjanjian terkait yang mengijinkan programmer aplikasi menggunakan interface tersebut sebagai bagian dari sebarang aplikasi yang terkait. Umumnya, sebuah application interface digunakan untuk mengakses sistem atau layanan jaringan yang sebaliknya akan memerlukan usaha pengembangan yang sama dengan perbuatannya. Sebagai contoh, Btrieve application interface mengijinkan programmer menggunakan struktur file Btrieve dan layanannya dalam aplikasinya. Lihat pula Application Programming Interface.
