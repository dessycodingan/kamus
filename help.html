Cara Menggunakan Aplikasi
  1. Pilihan menu yang diinginkan, aplikasi ini terdiri dari empat menu yaitu menu halaman awal, cari kata, tentang aplikasi, dan bantuan.

  2. Jika memilih Menu Cari Kata, maka user akan diarahkan untuk "mengetikan kata" pada kolom yang telah tersedia.

  3. Jika memilih Menu Tentang Aplikasi, maka user akan diberikan penjelasan kegunaan dari aplikasi ini.

  4. Jika memilih Menu Bantuan, maka user akan diberikan penjelasan bagaimana cara menggunakan aplikasi ini.
