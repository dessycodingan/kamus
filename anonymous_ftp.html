Anonymous FTP

Anonymous FTP memungkinkan seorang user untuk memanggil dokumen, file, program dan data-data lain dari tempat manapun dalam internet, tanpa harus menentukan user id dan password. Dengan menggunakan user id “anonymous” khusus, seorang user suatu jaringan akan langsung melewati pengujian keamanan lokal dan akan memiliki hak akses ke file-file yang dapat diakses secara umum pada remote system. Lihat juga File Transfer Protocol.
