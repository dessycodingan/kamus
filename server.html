Server

Suatu computer pada jaringan yang memiliki kemampuan pengenalan dan dapat menjawab layanan-layanan yang diminta pengguna. Layanan ini dapat memiliki range dari file dasar dan layanan printer hingga layanan untuk aplikasi lengkap terdistribusi. Sebagai contoh, sistem menajemen terdistribusi dapat membuat database logik tunggal melalui banyak server.
