Finger

Suatu program yang menampilkan informasi tentang user tertentu atau semua user yang sedang login pada local system atau pada remote system. Program ini pada umumnya menampilkan nama panjang user, waktu login terakhir, waktu idle, terminal line dan lokasi terminal (yang dapat dipakai). Program ini dapat juga menampilkan rencana dan project file yang ditinggalkan oleh seorang user.
