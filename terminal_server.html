Terminal Server

Suatu device yang menghubungkan banyak terminal ke LAN melalui sebuah hubungan jaringan. Terminal server juga menghubungkan banyak user jaringan ke port-port asynchronous-nya untuk kemampuan dial-out serta untuk mengakses printer.(Lihat juga Local Area Network)
